# UHS_in_a_depleted_gas_reservoir

This repository contains the simulation input files used in the study:

#### Title: Quantifying Operational Controls on Hydrogen Recovery Factor in Depleted Gas Reservoir for Underground Hydrogen Storage
#### Authors: Minkyung Chae, Tea-woo Kim, Suryeom Jo, Hung Vo Thanh, and Baehyun Min
#### DOI: 

#### Software
- Simulator: CMG-GEM 2924.10
- EOS: Peng-Robinson Equation of State

#### Repository Contents 
- Base_case.dat: Base-case simulation model
- Case_21.dat: Case 21 in the manuscipt, with the highest recovery factor among the investigated scenarios
- Include files.zip: Required include files that define the reservoir structure

#### Instructions 
1) Download all files from this repository
2) Extract INCLUDE FILE.zip
3) Place all extracted files in the same working directory as the .dat files
4) Open the desired simulation model in CMG-GEM and run the simulation
